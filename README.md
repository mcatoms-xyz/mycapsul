# MyCapsul

Your life, one dashboard, any AI.

A personal life dashboard that your AI maintains through conversation. Built for [Claude Desktop](https://claude.ai/download) with Cowork mode.

## Install

1. Download the latest `.plugin` file from [Releases](https://github.com/mcatoms-xyz/mycapsul/releases)
2. Open it with Claude Desktop
3. Say "set up my Capsul"
